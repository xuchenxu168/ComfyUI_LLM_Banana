# 编辑-Sora动漫5宫格提示词模板 使用指南

## 📖 模板概述

"编辑-Sora动漫5宫格提示词模板" 是一个为Sora模型设计的五宫格动漫漫画/短视频提示词生成模板。相比3宫格，5宫格提供了更完整的故事叙述空间，能展现更复杂的情节发展和角色互动。**总时长严格控制在25秒**。

## 🎯 核心特性

✅ **完整故事弧线** - 从开场到结局的完整叙述
✅ **精确时间控制** - 4+5+5+5.5+5.5秒 = 25秒总时长
✅ **角色一致性** - 跨5个镜头保持角色特征
✅ **专业节奏感** - 递进式故事推进
✅ **视觉多样性** - 5种不同的镜头类型

## 📥 输入参数

| 参数 | 说明 | 类型 |
|------|------|------|

## 📌 五幕故事结构

### Shot 1 (4秒) - 故事开场
**功能**：建立世界观、介绍环境、铺垫气氛
- 使用 **Wide Shot** 展现环境
- 引入男主角，建立场景氛围
- 快速吸引注意力

### Shot 2 (5秒) - 男主角发展
**功能**：展示主角的行动、情感或准备
- 使用 **Medium Shot + Movement**
- 深化主角人设
- 为后续冲突做准备

### Shot 3 (5秒) - 女主角登场
**功能**：女主角出现、首次互动、情感转折
- **转向性镜头**，改变故事方向
- 两个主角的首次互动
- 故事的转折点

### Shot 4 (5.5秒) - 关键互动
**功能**：两人的主要互动、冲突或和解
- 使用 **Close-up or Over-the-shoulder**
- 故事主要冲突/互动
- 情感接近高潮

### Shot 5 (5.5秒) - 故事高潮与结局
**功能**：情感高潮、结局揭示、余韵留白
- **高潮释放**镜头
- 情感达到顶峰
- 故事完成并留下余韵
