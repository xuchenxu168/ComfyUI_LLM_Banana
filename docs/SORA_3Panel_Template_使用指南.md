# 编辑-Sora动漫3宫格提示词模板 使用指南

## 📖 模板概述

"编辑-Sora动漫3宫格提示词模板" 是一个专门为Sora模型设计的三宫格动漫漫画/短视频提示词生成模板。它能够根据提供的角色参考图和风格参考图，生成高质量的三分格（3 panel）故事漫画提示词。

## 🎯 核心特性

✅ **角色一致性保持** - 在所有镜头中保持角色特征、表情和气质
✅ **视觉风格统一** - 根据参考图3的风格指导所有视觉元素  
✅ **电影级镜头语言** - 使用专业摄影术语和相机技术
✅ **叙事连贯性** - 三个镜头间故事推进自然流畅
✅ **详细的Sora提示词** - 生成符合Sora模型的详细英文提示词

## 📥 输入参数

### 必需参数
| 参数 | 说明 | 类型 |
|------|------|------|
| user_prompt | 三个镜头的脚本或场景描述 | STRING |
| template_type | 选择 "编辑-Sora动漫3宫格提示词模板" | 下拉选择 |
| model | 选择Gemini模型 | 下拉选择 |
| api_provider | API提供商 | 下拉选择 |

### 可选的参考图片
| 参数 | 说明 | 类型 |
|------|------|------|
| image | 参考图1 - 男主角参考 | IMAGE |
| image_2 | 参考图2 - 女主角参考 | IMAGE |
| image_3 | 参考图3 - 风格参考（画面风格、配色、光影等） | IMAGE |

## 🎬 三幕结构

### Shot 1 (4.5秒) - 场景建立
- **目的**：建立氛围、介绍环境、引入男主角
- **焦点**：男主角登场，设定情境

### Shot 2 (5.5秒) - 情感铺垫
- **目的**：女主角出现、情感反应、动作表现
- **焦点**：女主角登场，铺垫情感

### Shot 3 (5.0秒) - 关键互动
- **目的**：两人互动、关键时刻、情感转折
- **焦点**：双人互动，情感高潮

## 🎨 关键参数详解

### 相机运动术语
- **Dolly In/Out**: 推进/拉开
- **Tracking Shot**: 跟随拍摄
- **Pan**: 摇镜
- **Crane**: 升降
- **Static**: 静镜

### 提示词包含内容
- 角色特征和表情（与参考图一致）
- 场景环境描写
- 服装和配饰细节
- 光影和配色
- 相机技术参数
- 角色运动和姿态

## ⚙️ 使用步骤

1. **选择模板**：在 template_type 下拉中选择 "编辑-Sora动漫3宫格提示词模板"
2. **准备参考图**：
   - image: 男主角参考
   - image_2: 女主角参考
   - image_3: 风格参考
3. **输入脚本**：在 user_prompt 中输入三镜头的脚本描述
4. **配置模型**：选择合适的Gemini模型和API提供商
5. **生成提示词**：执行节点，获取生成的三宫格提示词
